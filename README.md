# TeachHub SaaS

A complete education platform for teachers to manage lesson planning, student rosters, assignments, and grading.

## Features

- 📚 Lesson Planning
- 👥 Student Management
- 📝 Assignment Tracking
- 📊 Grade Management
- 💳 Stripe Subscriptions
- 🔐 JWT Authentication

## Tech Stack

- **Frontend**: React 18
- **Backend**: Express.js
- **Database**: PostgreSQL
- **Auth**: JWT + bcryptjs
- **Payments**: Stripe
- **Hosting**: Vercel (frontend) + Railway (backend)

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install && cd server && npm install && cd ../client && npm install && cd ..`
3. Create .env files (see server/.env.example)
4. Start servers: `npm run dev`
5. Open http://localhost:3000
